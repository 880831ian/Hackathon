--------------------------------------------------------------
謝宗勳 nilson03137@gmail.com

Apache License 2.0
--------------------------------------------------------------
想解決被居家隔離的人，能夠透過這個防疫聊天室來與人連接感情，而不會被隔離就
失去與人交際的機會。
且現在假資料太氾濫，我們的主頁面有完整的衛服部相關資料釐清使用者的資訊。
提醒使用者須完成的防疫檢疫流程，例量體溫。
可防止需檢疫人員離開檢疫區域。
